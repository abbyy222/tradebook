Put your dashboard market photos in this folder with these exact names:

- `dashboard-market-1.jpg`
- `dashboard-market-2.jpg`
- `dashboard-market-3.jpg`

Use wide, high-quality market photos. The dashboard hero will crop them into a banner and apply the app's existing dark/terra/gold overlays automatically.

Best photo tips:

- Use street or market-wide scenes, not close-up portraits.
- Keep the image at least `1600px` wide if possible.
- JPG works best for photos.
- If you want fewer slides, you can reuse the same photo in more than one filename for now.

This folder powers the dashboard market carousel only.
